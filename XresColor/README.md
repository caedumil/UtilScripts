# GetXresColor

One-liner put on a script to get one of the colors set on `.Xresources`

## Usage

    getXresColor <color>

_color_ can be either a _name_ or a _code_ from the following table:

|   _name_          |   _code_  |
| ----------------- | --------- |
|   black           |   color0  |
|    red            |   color1  |
|   green           |   color2  |
|   yellow          |   color3  |
|   blue            |   color4  |
|   magenta         |   color5  |
|   cyan            |   color6  |
|   white           |   color7  |
|   brightblack     |   color8  |
|   brightred       |   color9  |
|   brightgreen     |   color10 |
|   brightyellow    |   color11 |
|   brightblue      |   color12 |
|   brightmagenta   |   color13 |
|   brightcyan      |   color14 |
|   brightwhite     |   color15 |
|   foreground      |           |
|   background      |           |
