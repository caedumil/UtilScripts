# Power Scripts

All scripts use `rofi` to present a list of options to the user.


## power.sh

- lock -> the screen
- sleep -> suspend to RAM
- hibernate -> suspend to disk
- logout -> logout from current session
- reboot -> reboot the machine
- poweroff -> shutdown the machine


## session.sh

- Toggle ScrenSaver
- Reload Session
- Reload HotKeys
- Reload GTK


## session.py

- ScrenSaver
- Bluetooth
- Reload Session
- Reload HotKeys
- Reload GTK
